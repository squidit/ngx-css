📘 Guia de Migração

Migrando de componentes legados para os novos componentes baseados em Reactive Forms

🔴 Forma Antiga (sq-input)

<sq-input
  [label]="'Email'"
  [required]="true"
  [(value)]="email"
></sq-input>

🟢 Forma Nova (sq-input-form-control)

<sq-input-form-control
  [label]="'Email'"
  [formControl]="emailControl"
  sqValidation
  [fieldName]="'Email'"
></sq-input-form-control>

// TypeScript
emailControl = new FormControl('', [
  Validators.required,
  Validators.email
]);

⚡ Performance
