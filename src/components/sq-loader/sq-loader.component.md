# Loader-Squid

## Inputs/Props

### 1. size (optional)

- description: Set the width and height of the loader
- type: `string`

### 2. color

- description: Set the color
- type: `string`
- default: `pink`

### 3. customClass (optional)

- description: Set Custon classes
- type: `string`

### 4. borderSize

- description: Set border-width loader
- type: `string`
- default: `0.20em`

## Outputs

## Example

```html
<sq-loader></sq-loader>
```
